# Documentação - Materiais Didáticos (12.º Ano)

Conjunto de materiais para aprender a explicar projetos fullstack com clareza: o que foi construído, como funciona, que decisões foram tomadas e como outra pessoa pode estudar ou continuar o trabalho.

## Pré-requisitos

-   Ter um projeto React + Node + MongoDB a funcionar.
-   Saber correr o projeto localmente.

## Antes de entregar um projeto

- [ ] README completo e setup funcional
- [ ] API documentada (exemplos + erros)
- [ ] Dados documentados (campos/validações/índices)
- [ ] Testes mínimos descritos e como correr
- [ ] Deploy explicado, com ambientes, logs e backups descritos
- [ ] Documentação para IA (contexto e limites)
