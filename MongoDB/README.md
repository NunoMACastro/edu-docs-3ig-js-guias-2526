# MongoDB - Materiais Didáticos (12.º Ano)

Conjunto de materiais em Markdown para aprender MongoDB com foco prático em APIs Node/Express e integração fullstack.

## Como usar

- Começa no ficheiro `01_introducao_e_setup_atlas.md` e segue a ordem.
- Faz os exemplos em pequenos projetos e valida os dados no Atlas.
- Os blocos [EXTRA] são opcionais para aprofundar.

## Pré-requisitos

- **JavaScript:** fundamentos concluídos.
- **Node.js + Express:** módulo Node até rotas/validação.
- **Conta MongoDB Atlas:** cluster e utilizador de BD.
- **Terminal + editor:** para correr scripts e API local.

## O que vais aprender

- Modelação de documentos e relações no modelo documento.
- CRUD com filtros, projeções e paginação.
- Uso equilibrado de **Node driver** e **Mongoose**.
- Validação, tratamento de erros por `id` e resposta consistente.
- Noções de desempenho com índices e pipelines de agregação.
